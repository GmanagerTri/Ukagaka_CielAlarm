■概要
このゴーストは『イオン de アラーム ～シェルノサージュ～』の機能を拡張し、PCで使えるようにするゴーストです。
アプリ版のあまりにも少ないアラームセット数を解決したい方や、PCでもイオンちゃんボイスのアラームを使いたい方向けです。

ただし、バグやエラーが出る可能性は未知数なため、重要な時には使用しないようにして下さい。
このゴーストを使用したことによるあらゆる損害・損失について、製作者は一切責任を負いません。

■導入
1.音声をmp3で録音
・PCのマイク端子とスマホをケーブルで繋げて録音する
・PC上のAndroidエミュレータから録音する
・PCのマイクにスマホのスピーカーを押し当てる
など、お好みの方法で録音して下さい。mp3以外で録音しても構いませんが、必ずmp3に変換して下さい。

音声ファイルの名前は、アプリ上のボイス選択のメニューで上から順番に01.mp3、02.mp3…50.mp3となるようにして下さい。
【例】
「おはよう、朝だよ！」の音声ファイル → 「01.mp3」
「ほら、起きて」の音声ファイル → 「02.mp3」

※音質が悪いのは機器の不調などではなく、大抵は元の音質が非常に悪いためです。諦めましょう。
※音声が始まる前に無音の時間があると、イマイチな出来になってしまいます。

2.音声ファイルをゴーストのフォルダにコピー
録音したファイルをゴーストのフォルダ（SSPのフォルダ/ghost/CielAlarm/ghost/master/）にコピーします。
すると、descript.txtやdic_menu.txtといったファイルと一緒に音声ファイル50個が並んでいるはずです。

3.起動してイオンちゃんの声が流れるか確認する
流れなかった場合、以下の点を確認して下さい。
・音声ファイルがしっかり.mp3形式になっているか（.wavなどの拡張子を.mp3に変えただけでは動きません）
・名前が01.mp3…になっているか

■使い方
・アラーム
1.メニューの「アラームセット」を選択します。
2.セットしたいアラームを押し、鳴らしたい曜日（月～日、平日、休日、毎日）と時刻をセットします。
3.起動中に限り、指定した時刻にアラームが起動します。（鳴らす時刻に応じたボイスが再生されます）
※アラームを削除したい場合は、削除したいアラームを選択し、「アラームを消去」を押して下さい。

・タイマー
1.メニューの「タイマーセット」を選択します。
2.セットしたいタイマーを選択します。
3.起動中に限り、時間になるとボイスが再生されます。

・喋る頻度を変更
自動的にボイスを再生する頻度を選択できます。
黙るを選択すると再生されなくなります。

・時間によるボイス再生
ONにすると丁度の時間にボイスが再生されます。

・繰り返し回数の設定
アラームを何回繰り返すか設定できます。
一度アラームが鳴り始めると、3つのセリフを設定した回数だけ呼びかけた後に、停止します。

・時計合わせ
PCの時計を合わせます。ユーザーアカウント制御が表示された場合、「はい」を選んで下さい。
